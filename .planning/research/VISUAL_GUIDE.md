1. User taps "Start" on centered Intro screen
2. Content fades out from center
3. **JUMP!** Game screen content appears ~100px lower
4. User experiences disorienting visual shift

### Root Causes

1. **Different layout strategies:**
   - 7 screens: `items-center justify-center` (true center)
   - 1 screen: `pt-16` + absolute HUD (offset from top)

2. **Viewport height issues (mobile web only):**
   - `100vh` includes space under browser toolbars
   - When toolbars hide, content shifts
   - `100dvh` fixes this
