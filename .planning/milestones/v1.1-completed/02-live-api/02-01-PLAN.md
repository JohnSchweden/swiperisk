Implementation must include:

2. Implement process() method that fills output with buffered audio
3. Implement addChunk() method receiving PCM data from main thread via MessagePort
4. Convert Int16 to Float32: sample / 32768.0
5. CRITICAL: Resample 24kHz→48kHz using linear interpolation:
   ```javascript
   function resample24to48(input) {
     const output = new Float32Array(input.length * 2);
     for (let i = 0; i < output.length; i++) {
       const srcIndex = i / 2;
       const srcIndexInt = Math.floor(srcIndex);
       const frac = srcIndex - srcIndexInt;
       output[i] = input[srcIndexInt] * (1 - frac) + 
                   input[Math.min(srcIndexInt + 1, input.length - 1)] * frac;
     }
     return output;
   }
   ```
6. Handle buffer management to prevent memory leaks:
   - Maximum buffer size (e.g., 10 chunks)
   - Clear processed chunks
7. Use MessagePort to receive audio chunks from main thread
