Modify CardStack.tsx to display incident images and preload next card:

1. Import dependencies:
   - import { ImageWithFallback } from '../ImageWithFallback'
   - import { getIncidentImagePath } from '@/data/imageMap'
   - Slugify helper: either import from imageMap or create inline helper
     (e.g., `const slugify = (s: string) => s.toLowerCase().replace(/[^a-z0-9]+/g, '-').replace(/(^-|-$)/g, '')`)

2. Find the current card body section (around line 254):
   ```
   <div className="p-4 md:p-10 flex flex-col justify-between flex-1 overflow-hidden">
     <div className="space-y-3 md:space-y-6 overflow-y-auto" ...>
       {/* Sender info currently starts here */}
   ```

3. Insert ImageWithFallback BEFORE the space-y-3 div (hero image at top):
   ```tsx
   <div className="p-4 md:p-10 flex flex-col justify-between flex-1 overflow-hidden">
     {/* Hero image - moves with card during swipe */}
     {currentCard.realWorldReference?.incident && (
       <div className="mb-4 md:mb-6 shrink-0">
         <ImageWithFallback
           src={getIncidentImagePath(slugify(currentCard.realWorldReference.incident))}
           alt={`Incident: ${currentCard.context}`}
           aspectRatio="video"
           containerClassName="max-h-[200px] md:max-h-none"
         />
       </div>
     )}

     <div className="space-y-3 md:space-y-6 overflow-y-auto" ...>
       {/* Existing sender info */}
   ```

4. Add preload for next card's image (eliminates placeholder flash on swipe):
   - Determine the next card in the deck
   - If it has a realWorldReference.incident, render a preload link:
   ```tsx
   {nextCard?.realWorldReference?.incident && (
     <link
       rel="preload"
       as="image"
       href={getIncidentImagePath(slugify(nextCard.realWorldReference.incident))}
     />
   )}
   ```
   - Place this in a Helmet/portal to <head>, or use useEffect with DOM manipulation
   - If using react-helmet isn't available, use a simple useEffect:
   ```tsx
   useEffect(() => {
     if (!nextCard?.realWorldReference?.incident) return;
     const link = document.createElement('link');
     link.rel = 'preload';
     link.as = 'image';
     link.href = getIncidentImagePath(slugify(nextCard.realWorldReference.incident));
     document.head.appendChild(link);
     return () => { document.head.removeChild(link); };
   }, [nextCard]);
   ```

5. Key constraints:
   - Image must be INSIDE the transformed container (the div with transform: translateX)
   - Use shrink-0 to prevent image from shrinking
   - Use max-h-[200px] on mobile to prevent excessive scrolling
   - Alt text: "Incident: {context}" for accessibility
   - Only render image when realWorldReference.incident exists

6. IMPORTANT: Image lookup is by slugified incident name, NOT card ID:
   - `getIncidentImagePath(slugify(currentCard.realWorldReference.incident))`
   - NOT `getIncidentImagePath(currentCard.id)`
